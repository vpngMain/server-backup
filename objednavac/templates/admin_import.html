Import

Import produktů

Soubor s hlavičkou. CSV nebo XLS/XLSX. Očekávané sloupce: název_skupiny | název | ks (jednotka) | pc. Volitelně sku.

Import interních produktů

CSV se stejnou hlavičkou jako běžné produkty: název_skupiny | název | ks | pc. Volitelně sku. Data jdou do samostatné tabulky interních produktů (kancelář). Přejít na přehled interních produktů.

Import objednávek

CSV s hlavičkou. Sloupce: branch_code (kód pobočky), product_id (ID produktu), quantity (množství). Řádky se stejným branch_code tvoří jednu objednávku. Pobočka musí existovat. Validace: pobočka, produkt, množství > 0.

Import EAN z katalogu

Přiřazení EAN k produktům podle katalog.xlsx (sloupce nazev, kod_zbozi, ean). Otevřít import EAN z katalogu.

Import výdejek

CSV s hlavičkou. Sloupce: order_id, order_item_id, shipped_quantity. Aktualizuje odeslané množství u položek objednávky. Validace: objednávka a položka musí existovat.
